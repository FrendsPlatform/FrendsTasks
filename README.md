# Frends Tasks

This repository serves as a central place for sharing Frends Tasks related things. Currently, it means:

- [Workflows](/.github/workflows) – GitHub Actions workflows files used by all Frends Tasks.
- [Analyzers](/FrendsTaskAnalyzers) - Coding standards analyzers for Frends Tasks.
- [Toolkit Template](/FrendsToolkitTemplate) – Template to create a Toolkit solution for reusable code in a system repository.
- [Task Template](/FrendsTaskTemplate) – Template to create a new Frends Task solution.

See [Frends iPaaS](https://frends.com) for more information on the product.
